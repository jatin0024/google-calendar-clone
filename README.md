#  Google Calendar Clone

A responsive **Google Calendar Clone** built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Drizzle ORM** using **PostgreSQL** (works with Neon or local Postgres). Add, edit, and view events in day/week/month views.

---

##  Features
- Create, edit, and delete events  
- Day / Week / Month calendar views  
- Type-safe backend with Drizzle ORM  
- Responsive UI with Tailwind CSS  
- Compatible with Neon (serverless Postgres) or local PostgreSQL

---

##  Tech Stack
- Next.js (TypeScript)  
- Tailwind CSS  
- Drizzle ORM + drizzle-kit  
- PostgreSQL (Neon-compatible)  
- Utilities: dayjs, Zustand, Radix UI
